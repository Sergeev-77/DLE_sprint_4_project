# Проект модуля. Нейросеть для предсказания калорийности блюд

По фотографии, ингредиентому составу и массе блюда предсказываем его калорийность

## Структура проекта 
* dataset.py - подготовка данных, аугментация
* utils.py - структура модели, основные функции по обучению, валидации и визуализации процесов обучения и инференса
* solution.ipynb - базовая конфигурация проекта,  итоги изучения данных, обучения и инференса модели
